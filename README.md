# 🐍 Snake Game using Python Tkinter

A classic Snake Game developed using Python and Tkinter. The player controls a snake that grows longer by eating food. The game ends when the snake collides with the wall or its own body.

## 🎮 Features

* Interactive GUI using Tkinter
* Score tracking system
* Random food generation
* Snake grows after eating food
* Collision detection with walls
* Self-collision detection
* Game Over screen with final score

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI Library)
* Random Module

## 🚀 How to Run

### Prerequisites

Make sure Python 3 is installed on your computer.

Check installation:

```bash
python --version
```

### Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/snake-game.git
```

2. Navigate to the project folder:

```bash
cd snake-game
```

3. Run the game:

```bash
python snake_game.py
```

## 🎯 Controls

| Key           | Function   |
| ------------- | ---------- |
| ↑ Up Arrow    | Move Up    |
| ↓ Down Arrow  | Move Down  |
| ← Left Arrow  | Move Left  |
| → Right Arrow | Move Right |

## 🕹️ Gameplay

* Control the snake using the arrow keys.
* Eat the red food blocks to increase your score.
* The snake grows longer every time it eats food.
* Avoid hitting the walls.
* Avoid colliding with your own body.
* The game ends when a collision occurs.
